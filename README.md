Travel_Booking_System

A digital platform to manage trip reservations efficiently, including payments and cancellations. The system helps users book and cancel trips, handle payments, apply cancellation policies, and track loyalty points. It also allows filtering trips by destination or date and securely saves booking information.

Features
- Book and cancel trips
- Handle payments
- Apply cancellation policy
- Filter trips by destination/date
- Track loyalty points
- Save booking info


Classes

1-Customer
Represents a user of the system with personal details and booking history.

2-Trip
Represents a travel trip with details such as destination, date, price, and availability.

3-Booking
Handles information about a specific booking, including customer and trip details.

4-BookingManager
Manages the creation, modification, and cancellation of bookings.

5-PaymentManager
Handles all payment processing and validation for trips.

6-CancellationManager
Applies cancellation policies and processes refund or penalty rules.

7-FileManager
Manages saving and retrieving booking information from files.
